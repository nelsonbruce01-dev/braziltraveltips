# Brazil Travel Tips

Premium travel guidance for international visitors planning Brazil, with an English-first editorial structure and future multilingual expansion.

## Cloudflare Workers Static Assets

Cloudflare deploys this repository automatically from GitHub.

- Deploy command: `npx wrangler deploy`
- Public assets directory: `site/`
- Wrangler assets config: `directory = "./site"`

The public website lives inside `site/` so repository metadata is not published.
